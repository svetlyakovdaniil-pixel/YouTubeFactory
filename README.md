# YouTube Factory

Автоматическая система для генерации, сборки и публикации музыкальных видео и трансляций на YouTube.

## Возможности

- автоматическая сборка видео
- автоматическая анимация изображений
- автоматическое создание плейлистов
- автоматический запуск трансляций
- поддержка нескольких YouTube-каналов
